# book-server

misikt 教师端复刻 — **后端服务**。

## 项目说明

功能级复刻 [misikt 教师端](https://www.misikt.com/teacher/) 的 **题库 + 卷库** 两个核心模块的业务流。

- **范围**：题库 + 卷库 only；学生 / 班级 / 批改 / 统计 不做
- **认知工程**：[misworkplace](https://github.com/RuntimeAll/misworkplace) — 项目认知层 + 任务卡 + 数据建模
- **前端**：[book-ui](https://github.com/RuntimeAll/book-ui)

## 状态

🟢 基建中（V0.0）— 待初始化工程结构。
